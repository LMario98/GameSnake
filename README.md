# 🐍 Snake Game

Un classico gioco Snake costruito da zero con HTML, CSS e JavaScript puro — nessuna libreria esterna.

## Demo



## Funzionalità

- Movimento fluido con frecce direzionali o WASD
- Punteggio in tempo reale
- Record salvato nel browser (localStorage)
- Rilevamento collisioni con i bordi e il corpo
- Design dark theme
- Responsive sul desktop

## Tecnologie usate

- HTML5 Canvas API
- CSS3
- JavaScript ES6


Nessuna installazione o dipendenza richiesta.

## Struttura del progetto
```
snake-game/
├── snake.html   # struttura della pagina
├── style.css    # design e dark theme
├── game.js      # logica del gioco
└── README.md    # documentazione
```

## Cosa ho imparato

- Usare la Canvas API per disegnare e animare elementi
- Gestire un game loop con `setInterval`
- Lavorare con array e oggetti in JavaScript
- Salvare dati nel browser con `localStorage`
